## Детализация критичных находок

### TASK-0006: mypy ошибки (64 шт)
**Проблемы:**
- `union-attr`: доступ к атрибутам `Message|InaccessibleMessage|None` без проверки на None
- `type-arg`: `dict` без параметров типов (должен быть `dict[str, Any]`)
- `assignment`: присваивание `Wallet|None` переменной типа `Wallet`
- `arg-type`: передача `dict[Any, Any]` вместо `AsyncSession` в репозитории

**Файлы:** `src/wrbot/bot/handlers/wallets.py`, `src/wrbot/bot/handlers/categories.py`, `src/wrbot/bot/handlers/settings.py`, `src/wrbot/bot/handlers/start.py`, `src/wrbot/bot/keyboards.py`

**Примеры:**
```python
# wallets.py:28 - wallet_id может быть None
wallet_id = int(callback.data.split("_")[1])  # Может упасть

# wallets.py:56 - session из dict[Any, Any] вместо AsyncSession
repo = WalletRepository(session)  # type error

# categories.py:142 - wallet может быть None
wallet = await repo.get(...)
wallet_name = wallet.name  # union-attr
```
