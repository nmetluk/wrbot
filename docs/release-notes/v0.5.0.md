# wrbot v0.5.0 — Валюты списаний

По просьбам пользователей списания теперь поддерживают валюты: выбор при создании, встроенный
справочник всех валют и отображение сумм в нужной валюте вместо всегда «₽».

## ✨ Что нового

- **Выбор валюты при создании списания.** Быстрые кнопки **RUB / EUR / USD / KZT / KGS** и кнопка
  **«🌐 Другая валюта»** — постраничный список всех валют + поиск по коду или названию
  (например «GBP» или «pound»).
- **Запоминание валюты.** Бот предлагает последнюю использованную вами валюту по умолчанию
  (для новых пользователей — RUB) — обычно выбирать заново не нужно.
- **Встроенный справочник ISO 4217.** В проект добавлен оффлайн-справочник (153 активные валюты),
  выбор работает без обращения к сети.
- Валюту можно изменить при редактировании списания.

## 🔧 Изменения

- **Суммы отображаются в валюте списания** везде — при создании, в «Мои списания», в карточке и в
  напоминаниях. Для популярных валют показывается символ (₽, €, $), для остальных — трёхбуквенный код.

## 🛠 Технологии

Python 3.11+ · aiogram 3.x · SQLAlchemy 2.0 (async) + Alembic · APScheduler ·
SQLite (старт) / PostgreSQL.

## 🚀 Обновление

```bash
git pull
docker compose up -d --build   # применит миграцию валютных полей автоматически
```

Миграция добавляет `charges.currency` (с дефолтом RUB и backfill существующих) и
`users.last_currency`. **Перед обновлением на проде — свежий бекап БД** (см. `docs/deployment.md`).

## ✅ Качество

- 263 теста; полный CI на зафиксированных версиях (`uv.lock`): `ruff`, `mypy --strict`, `pytest`,
  миграции Alembic (проверены `alembic check` и round-trip), валидация состояния.
- Добавлены router-level тесты выбора валюты. Принят аудитом майлстоуна
  (`handoff/reports/AUDIT-M8-2026-06-04.md`).

## ⚠️ Примечания

- Конвертация между валютами не выполняется — валюта это атрибут суммы списания.
- Названия валют в справочнике — на английском (RU-локализация названий — возможна позже).
