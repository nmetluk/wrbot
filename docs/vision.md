# Видение продукта

## Проблема
Люди забывают о регулярных и разовых списаниях (подписки, ЖКХ, кредиты) и
сталкиваются с просрочками, штрафами и нехваткой средств на нужной карте.

## Решение
Лёгкий Telegram-бот, который ведёт персональный список будущих платежей и
заранее (за 5/3/1 день — настраивается) напоминает о них в личном чате, с
быстрыми действиями «Оплачено / Отложить / Редактировать».

## Для кого
Частный пользователь Telegram, ведущий личные финансы «в голове» и желающий
простой бесплатный инструмент-напоминалку без сложной бухгалтерии.

## Ценность и принципы
- **Простота прежде всего.** Inline-кнопки и пошаговые диалоги, минимум ввода руками.
- **Приватность по умолчанию.** Данные изолированы по `tg_id`, ничего лишнего не собираем.
- **Надёжность.** Бот работает 24/7, напоминания приходят вовремя (±1 минута).
- **Расширяемость.** Архитектура готова ко второй итерации без переписывания.

## Границы первой версии (scope)
**Входит:** CRUD списаний, кошельков, категорий; пошаговое создание списания;
список ближайших платежей; «отметить оплаченным» со сдвигом даты по периоду;
глобальные и индивидуальные настройки уведомлений; русский интерфейс.

**Не входит (вторая итерация):** экспорт CSV/PDF, статистика, мультивалютность,
семейный/групповой режим. См. ТЗ §6.

## Критерии успеха v1
1. Пользователь за < 1 минуты добавляет списание и получает напоминание в нужный день/время.
2. «Оплачено» корректно сдвигает дату по периодичности.
3. Бот стабильно работает 24/7, не теряет и не дублирует напоминания.
